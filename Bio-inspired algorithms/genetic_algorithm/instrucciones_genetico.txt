* Variables que se pueden dar por linea de comando:

--doc "archivo_ovejas.txt" -> Se le pasa la dirección donde se encuentra el archivo con la información de las ovejas, si no existe se genera el mismo con 10 000 ovejas predeterminadas.

--verb 1 -> Puede tomar valores de 1 ó 0 para mostrar cambios en el algoritmo

--gens -> Numero de generaciones maximas

--peso -> Peso maximo para el problema

--pob  -> Tamaño de la población

* Ejemplo para ejecutar el programa con documento "ovejas.txt", con peso 200:

	python3 ovejas_algoritmo_genetico.py --doc ovejas.txt --peso 200
